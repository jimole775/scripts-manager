# 自动编辑脚本
## 功能描述
视图展示串行的流程节点图，每个节点表示一个点击操作，节点之间用箭头连接，表示点击操作的顺序
编辑好之后，点击保存按钮，将流程节点图保存到配置文件中。
并生成一个读取配置文件的脚本，用于在实际运行时读取配置文件并执行自动点击操作。

## 页面布局
1. 左侧是已保存的自动点击配置列表，每个配置项包含配置名称和配置文件路径
2. 中间是流程节点图的视图区域，用于展示和编辑点击操作的流程节点图
3. 右侧是节点属性和操作按钮区域，包含添加节点、删除节点、保存配置、执行脚本等按钮

## 操作步骤
1. 点击添加节点按，在视图中添加一个新的节点
2. 点击节点，在属性面板中可以编辑节点的属性，如等待时间，找不到图片是否持续等待等
3. 节点的连接线，可以删除
4. 每个节点可以上传一个图片
4. 每个节点左侧1个连接点代表入口；右边两个连接点，分别代表图片找到成功和失败。
4. 点击保存按钮，将流程节点图保存到配置文件中，并在呈现在左侧配置列表
5. 点击执行脚本按钮，根据配置文件生成一个读取配置文件的脚本，用于在实际运行时读取配置文件并执行自动点击操作。


## 脚本逻辑
1. 读取每个节点的配置，截取当前桌面的图片，然后使用节点中的图片，与截取的图片进行匹配，找到点击位置。
2. 如果找到点击位置，点击该位置，然后根据成功的连接线，继续读取下一节点。
3. 如果没有找到点击位置，然后根据失败的连接线，继续读取下一节点。
4. 如果没有失败的连接线，或者没有找到点击位置，脚本执行结束。

## 脚本规范
1. 介于当前的项目特点，挑选一种合适的脚本语言，如python、javascript等
